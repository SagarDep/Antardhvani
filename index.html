About this project.

This project is about the Univeristy of Delhi's annual festival, Antardhvnani's, official Android application. If you want to contribute to this project, or play around with its source code or maybe just just want to learn or take a glance at its internal working, clone the repository:

$ git clone https://github.com/teamDAPSR/Antardhvani.git

Or you can use Github for Windows/Mac if you want. Or, simply download the archive link on the top right.
